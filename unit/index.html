단위 변환 안내

길이 단위

mm (밀리미터): 1cm의 1/10, 정밀 측정에 사용됩니다. cm (센티미터): 일상에서 가장 많이 사용하는 길이 단위입니다. m (미터): 국제 표준 길이 단위, 1m = 100cm입니다. km (킬로미터): 장거리 측정에 사용, 1km = 1,000m입니다.

inch (인치): 1인치 = 2.54cm, TV나 모니터 크기에 사용됩니다. feet (피트): 1피트 = 12인치 = 30.48cm입니다. yard (야드): 1야드 = 3피트 = 91.44cm입니다. mile (마일): 1마일 = 1.609km, 미국에서 거리 단위로 사용합니다.

무게 단위

mg (밀리그램): 1g의 1/1000, 의약품 등 미량 측정에 사용됩니다. g (그램): 일상에서 가장 많이 쓰는 무게 단위입니다. kg (킬로그램): 1kg = 1,000g, 체중 등에 사용됩니다. ton (톤): 1톤 = 1,000kg, 대형 화물 등에 사용됩니다.

lb (파운드): 1파운드 = 약 453.6g, 미국에서 체중에 사용됩니다. oz (온스): 1온스 = 약 28.35g, 식품 등에 사용됩니다.

온도 단위

섭씨(°C): 물의 어는점 0°C, 끓는점 100°C를 기준으로 합니다. 한국을 포함한 대부분의 국가에서 사용합니다. 화씨(°F): 미국에서 주로 사용, 32°F = 0°C, 212°F = 100°C입니다. 켈빈(K): 과학에서 사용하는 절대온도, 0K = -273.15°C입니다.

넓이 단위

㎡ (제곱미터): 가로 1m × 세로 1m의 넓이입니다. 평: 한국 전통 넓이 단위, 1평 = 약 3.3058㎡입니다. acre (에이커): 1에이커 = 4,046.86㎡, 토지 면적에 사용됩니다. ha (헥타르): 1ha = 10,000㎡, 농지 면적에 사용됩니다.

부피 단위

ml (밀리리터): 1L의 1/1000, 음료 등에 사용됩니다. L (리터): 1L = 1,000ml, 액체 용량의 기본 단위입니다. gallon (갤런): 미국 갤런 = 약 3.785L, 휘발유 등에 사용됩니다. fl oz (액량 온스): 1fl oz = 약 29.57ml, 음료에 사용됩니다.

데이터 용량 단위

B (바이트): 컴퓨터 저장의 기본 단위, 8비트 = 1바이트입니다. KB (킬로바이트): 1KB = 1,024B, 작은 파일 크기입니다. MB (메가바이트): 1MB = 1,024KB, 사진이나 문서 파일 크기입니다. GB (기가바이트): 1GB = 1,024MB, 영상이나 앱 크기입니다. TB (테라바이트): 1TB = 1,024GB, 하드디스크 용량입니다.
