How to Use

Enter text to see the character count, byte count, word count, and line count calculated in real time.

Usage Examples

  • Checking character limits for blogs/social media (Instagram: 2,200, Twitter: 280)
  • Adjusting character count for resumes and cover letters
  • Checking byte count for SMS messages (Korean: 2 bytes, English: 1 byte)
  • Naver blog titles (40 characters), body summaries (200 characters), etc.

Byte Calculation Standards

Based on UTF-8, Korean characters are 3 bytes, while English/numeric characters are 1 byte. Based on EUC-KR, Korean characters are 2 bytes, while English/numeric characters are 1 byte. SMS and some systems use the EUC-KR standard.
